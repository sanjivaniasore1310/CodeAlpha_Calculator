## 📌 Project Overview

The **Responsive Calculator Web Application** is a front-end project built using **HTML, CSS, and JavaScript**.

It performs basic arithmetic operations including:

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)

The calculator features a modern user interface, real-time input display, and full keyboard support for enhanced usability.

This project demonstrates strong understanding of UI design, DOM manipulation, and event handling in JavaScript.
